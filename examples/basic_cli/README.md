# Seisei Basic CLI

Offline example that exercises public Seisei contracts with a deterministic fake provider.

Run:

```sh
dart run
```

This example does not call Apple Foundation Models, PCC, or any cloud provider.
For a live local AFM smoke on macOS 27, use `packages/seisei_apple` with the
`/usr/bin/fm` CLI or a Flutter host that imports the native bridge.
